# bems-front

## Project Setup

### Setup config

Fill the `.env` file with your own configuration.

### Install dependencies

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Run tests

```sh
npm run cypress:run
```
